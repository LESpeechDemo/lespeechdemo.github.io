LESpeech: Break Language Barriers for Cross-language Speaker-Adaptive Text-to-Speech

Anonymous Authors

1. Abstract

Zero-shot cross-language speaker-adaptive text-to-speech has advanced rapidly, enabling more controllable speech generation across different languages. However, existing approaches suffer from language coverage limitation, with state-of-the-art systems supporting only 0.5% of the world’s 7,000+ languages. One common method to expand language coverage is pre-training on a large number of languages upfront. However, this requires collecting extensive multilingual data simultaneously, which is particularly challenging for low-resource languages. Another approach is pre-training on several high-resource languages, followed by fine-tuning for new low-resource languages. However, this often leads to catastrophic forgetting and degrading performance in previously learned languages. To address these challenges, we propose Language Expert Speech (LESpeech), a zero-shot cross-language speaker-adaptive TTS framework that can seamlessly integrate unseen low-resource languages without fine-tuning the pre-trained modules. LESpeech achieves this by disentangling speech generation into two phases: the source phase, which uses language-specific experts to map the texts from different languages into a language-unified, speaker-independent phonetic feature space, and the filter phase, which applies the target speaker’s voice characteristics using a language-universal filter module to generate speech. This design enables efficient language expansion by training a new language expert for the target language with only one hour of paired audio-text data. Our experiments demonstrate that LESpeech, while leveraging only a minimal amount of data for language expansion, achieves up to a 5.5% improvement in naturalness and a 90% relative reduction in character error rate compared to existing work on ZCS TTS language coverage. Speech samples generated by LESpeech can be found at https://lespeechdemo.github.io/.





* In this demo, CER is computed by transcribing the audio using the MMS-1B-all ASR model (with language adapters), followed by calculation using jiwer with punctuation and capitalization removed.

Conclusion

LESpeech is the first ZCS TTS system explicitly designed to overcome language coverage limitations while producing intelligible and natural-sounding speech. After pre-training, LESpeech can be efficiently adapted to new low-resource languages using only one hour of paired text-speech data. This is achieved by decomposing the speech synthesis process into two stages: a language-independent content generation stage and a speaker-specific filtering stage. Experimental results demonstrate that LESpeech significantly outperforms previous approaches in addressing the language coverage challenge and achieves speech quality competitive with SOTA ZCS TTS models trained on large-scale datasets.
